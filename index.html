How to use the CSS

Grab the milligram.css file from this repo and link it in your HTML head. No images or other dependencies needed.

What's different from milligram

Tomato uses Milligram as a base, plus adds the following classes for you to use:

Container Classes

.hero
For an attention getting banner that goes all the way across the page
.big-section
Acts just like Milligram's .container, plus lots of added top and bottom margin for breathing room
.footer
For the page footer - has a gray bar appearance to mirror the top main menu.

Header Bar Classes

.main-menu
For a dark gray menu bar that goes all the way across the screen. Automatically becomes a flexbox container on medium-ish screens
.logo
For the logo. Usually found inside .main-menu
.menu-links
For the list of navigation links in the .main-menu

Example HTML:

<nav class="main-menu">
	<h1 class="logo"><a href="index.html">Tomato</a></h1>
	<ul class="menu-links">
		<li><a class="button button-clear" href="html-tags.html">HTML Tags</a></li>
		<li><a class="button button-clear" href="grid-layout.html">Grid Layout</a></li>
		<li><a class="button button-clear" href="index.html">Example Page</a></li>
	</ul>
</nav>

Breakpoints

Tomato adds "smallish," "mediumish," and "largish" breakpoints for use in sass customization

Example SCSS:

.main-menu{
	display:block;
	@include mediumish{
		display: flex;
	}
}
